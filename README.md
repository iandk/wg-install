# Wireguard Install
## Installation
Download
```
wget -qO /usr/local/bin/wg-install https://git.io/JedO6
chmod +x /usr/local/bin/wg-install
```

Open script and change `myvpn.domain.io`  your FQDN or IP
```
nano /usr/local/bin/wg-install
```
Run installer
```
wg-install install
```

## Usage


```
wg-install add
```
